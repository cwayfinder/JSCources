Why you want to try WebStorm

Code editor with an IQ of its own

The editor understands your code, its structure and offers you code completion that you expect. Full coding assistance is offered even for language mixtures such as HTML inside JavaScript strings.

Code quality analysis

Hundreds of language specific code inspections analyse the whole project as you type and show you the identified problems before you open the browser. Intelligent quick-fixes offer instant solutions.

Transparent VCS integration

Git, SVN, Mercurial are just the top 3 that WebStorm supports. The unified UI helps you do your job no matter where you keep your code. Local history keeps your files safe even if you use no VCS.

Endless productivity growth potential

Built-in tools like Terminal, REST Client and npm are at your fingertips for efficient development. You’ll love keyboard access to any IDE action, not to mention the killer Live Edit feature.

FAQ

  • What does your "per concurrent user" licensing mean?
    A single license permits the use of IntelliJ IDEA by a single user at a time. For multiple concurrent sessions you must purchase multiple licenses.
  • How does the IntelliJ IDEA license work?
    Each IntelliJ IDEA license is associated with a unique key-code registered under your user name that allows you to access the software.
  • My company will have multiple developers using IntelliJ IDEA. Should we license IntelliJ IDEA to each developer personally?
    If you are purchasing IntelliJ IDEA for a company, you should register all your licenses in the company name. That way, no personal registration is required and any employee or contractor can use one of your company-registered licenses. See also: Can we continue using the license of a developer who is no longer with our company?.
